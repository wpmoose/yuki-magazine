== Description ==

Yuki Magazine is based on the modern and highly customizable Yuki multipurpose WordPress theme. It is optimized for online magazines, newspaper, news portals, content publishers. You can fully customize your site with the built-in easy-to-use and intuitive Homepage Builder.
